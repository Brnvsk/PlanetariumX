Игра Карточки Памяти: Космос

Как играть? Как играть?

Эта игра показывает огромное множество космических тел, существующих в нашем уголке вселенной.Исследуйте пространство, не вставая с кресла. Ваша задача открыть все карточки. На них будут изображены астероиды, планеты, звезды, юные астронавты, космические станции и так далее. Вы должны открыть картинки, только обязательно две одинаковые, тогда они вернутся в прежнее положение.
